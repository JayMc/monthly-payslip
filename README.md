# monthly-payslip
MYOB coding challenge

Given salary and super rate calculates employee income tax, net income and super for 1 month.

# Get started
 1. git clone https://github.com/JayMc/payslip-calc.git
 2. npm install
 3. bower install
 4. node server.js

# Running tests
 1. npm install -g mocha
 2. mocha tests
